## 기능 목록

- input
1. 다리 길이를 입력 받는다.
    - 3~20 사이의 숫자이다.
2. 이동할 칸을 입력 받는다.
   - 위 칸 -> U, 아래 칸 -> D

- output
1. 다리 화면 출력
   - 이동한 칸을 건널 수 있다면 -> O 건널 수 없다면 X

- bridge
1. 다리 길이 만큼 다리를 생성한다.
   - List<String> 위 칸 -> U, 아래 칸 -> D
   - 랜덤 숫자  1 -> 위 칸, 0 -> 아래 칸

## 에러 처리
- 에러 발생시 [ERROR] 단어가 포함 되어있어야 한다.
- 그 부분 부터 다시 입력 받는다.

1. 다리 길이는 3~20 사이의 숫자이다.
2. 이동할 칸은 'U', 'D' 이다.
