# 기능 목록

## Input

- [x] 다리 길이를 입력 받는다.
- [ ] 이동할 칸을 입력 받는다.
- [ ] 게임 재시작/종료 여부를 입력 받는다.

## Bridge
- [ ] 다리 위 칸과 아래 칸중 건널 수 있는 칸을 0과 1중 무작위 값을 이용해서 생성한다.
- [ ] 재시작해도 처음에 만든 다리를 재사용한다.

## Count
- [ ] 시도한 횟수를 계산한다.
- [ ] 게임 성공 여부를 확인한다.

## Output
- [ ] 다리의 상태를 출력한다.
- [ ] 최종 게임 결과를 출력한다.
- [ ] 게임 성공 여부를 출력한다.
- [ ] 총 시도한 횟수를 츨력한다.

# 에러 처리
- [x] 다리 길이는 3~20 사이의 숫자이다.
- [ ] 이동할 칸은 U(위 칸) 이나 D(아래 칸) 증 하나의 문자를 입력받는다.
- [ ] 게임 재시작/종료 여부는 R(재시작) 이나 Q(종료) 중 하나의 문자를 입력 받는다.
